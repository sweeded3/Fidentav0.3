# Fidenta static site

Статический сайт для GitHub Pages.

## Структура

- `index.html` — главная страница
- `invest.html` — страница инвестиций
- `lk.html` — личный кабинет инвестора
- `privacy.html` — политика конфиденциальности
- `terms.html` — правовая информация
- `consent.html` — согласие на обработку персональных данных
- `.nojekyll` — служебный файл для GitHub Pages

## Публикация

Загрузить содержимое архива в корень репозитория и включить GitHub Pages для ветки `main` / root.

## Проверка step 10

Проверены локальные переходы, якоря, title/meta-description, кликабельные телефоны и почта, футеры и юридические ссылки.
